async: A palavra async antes de uma função significa que a função sempre retorna uma promise. Outros valores serão agrupados em uma promise resolvida automaticamente. Por exemplo, esta função async retorna uma promise resolved com a string “Ola Mundo”: async function funcaoAsync() {. return 'Ola Mundo';

E o await
O operador await é utilizado para esperar por uma Promise. Ele pode ser usado apenas dentro de uma async function.
